============================================
NEXUS - Student Innovation Club
Images Folder
Stage 2: Functionality
============================================

This folder is designated for storing all images, 
logos, and other media assets used in the website.

RECOMMENDED IMAGES TO ADD:
--------------------------

1. LOGO (logo.png or logo.svg)
   - Club logo in high resolution
   - Recommended size: 200x200 pixels minimum
   - Transparent background preferred

2. HERO IMAGES
   - hero-banner.jpg - Main homepage banner image
   - Recommended size: 1920x1080 pixels
   - Show students working on projects or at events

3. ABOUT PAGE IMAGES
   - about-team.jpg - Team photo
   - about-office.jpg - Innovation Hub photo
   - Recommended size: 800x600 pixels

4. EVENT IMAGES
   - event-hackathon.jpg
   - event-workshop.jpg
   - event-conference.jpg
   - Recommended size: 600x400 pixels

5. GALLERY IMAGES
   - gallery-01.jpg through gallery-12.jpg
   - Various event and activity photos
   - Recommended size: 800x600 pixels

6. TEAM MEMBER PHOTOS
   - team-member-01.jpg through team-member-04.jpg
   - Professional headshots
   - Recommended size: 300x300 pixels (square)

FILE NAMING CONVENTIONS:
------------------------
- Use lowercase letters
- Use hyphens (-) instead of spaces
- Use descriptive names (e.g., "hackathon-2024.jpg")
- Include file extensions (.jpg, .png, .svg)

SUPPORTED FORMATS:
------------------
- .jpg / .jpeg - For photographs
- .png - For images requiring transparency
- .svg - For logos and icons
- .gif - For animations (if needed)

NOTE: The website currently uses CSS placeholders and 
emoji icons. Replace these with actual images when available.
